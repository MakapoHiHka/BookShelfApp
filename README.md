# Личная библиотека

«Личная библиотека» — это серверное веб-приложение для управления коллекцией электронных книг и отслеживания личного прогресса чтения.
Приложение позволяет:
Загружать электронные книги (.epub) в свою библиотеку;
Читать книги через браузер с удобным разбиением на страницы (по 150 слов);
Автоматически сохранять место, на котором вы остановились;
Организовывать книги по именованным полкам;
Безопасно регистрироваться и входить в систему (с подтверждением email и защитой от ботов).
Вся логика реализована в виде RESTful API, что делает приложение полностью backend-ориентированным и готовым к интеграции с любым фронтендом или мобильным клиентом.
Система разработана с учётом безопасности, надёжности и удобства сопровождения.
Основное назначение — предоставить пользователю персональный, приватный и функциональный инструмент для работы с электронными книгами без зависимости от внешних сервисов.

## Основные возможности

- Регистрация и аутентификация (JWT + подтверждение email)
- Загрузка книг в формате `.epub`
- Автоматическое разбиение книги на страницы (по 150 слов)
- Сохранение прогресса чтения (номер текущей страницы)
- Организация книг по полкам
- Асинхронная обработка контента (`@Async`)
- reCAPTCHA на регистрации
- Полностью RESTful API

## Технологии

- **Backend**: Spring Boot 4, Java 17  
- **База данных**: PostgreSQL  
- **Безопасность**: Spring Security, JWT, BCrypt  

## Запуск

1. Убедитесь, что установлены:
  - JDK 17+
  - Maven
  - PostgreSQL

2. Настройте параметры в application.properties:  
  - Подключение к БД  
  - Ключи reCAPTCHA (также нужен ваш sitekey в register.html)  
  - Данные Hogmail (для email)  

3. Соберите и запустите

Приложение будет доступно по адресу: http://localhost:8080/bookShelf/register.html
